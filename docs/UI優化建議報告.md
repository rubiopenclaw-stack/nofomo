## 🎯 三大優化方向

### 1️⃣ 統一色彩系統

**現有問題**：
- 色彩過少，只有 4 個 CSS 變數
- 缺乏中性灰階層次
- 買入/賣出僅靠左邊框區分，不夠直覺

**建議改進**：
```css
:root {
  /* 主色調 - 金融科技感 */
  --primary: #4F46E5;
  --primary-light: #818CF8;
  --primary-dark: #3730A3;
  
  /* 漲跌色彩 - 標準金融綠/紅 */
  --gain: #10B981;
  --gain-light: #D1FAE5;
  --loss: #EF4444;
  --loss-light: #FEE2E2;
  
  /* 中性灰階 */
  --gray-50: #F9FAFB;
  --gray-100: #F3F4F6;
  --gray-200: #E5E7EB;
  --gray-300: #D1D5DB;
  --gray-500: #6B7280;
  --gray-700: #374151;
  --gray-900: #111827;
  
  /* 背景層次 */
  --bg-main: #F8FAFC;
  --bg-card: #FFFFFF;
  --bg-surface: #F1F5F9;
  
  /* 強調色 */
  --accent: #F59E0B;
  --info: #0EA5E9;
}
```

**具體改動**：
| 元素 | 現狀 | 建議 |
|------|------|------|
| 買入卡片 | 僅左邊框綠色 | 整張卡片微綠背景 + 綠色邊框 |
| 賣出卡片 | 僅左邊框紅色 | 整張卡片微紅背景 + 紅色邊框 |
| P&L 正數 | 文字綠色 | 綠色背景標籤 + 圖示 ▲ |
| P&L 負數 | 文字紅色 | 紅色背景標籤 + 圖示 ▼ |

### 2️⃣ 損益儀表板改善

**現有問題**：
- 統計頁只有 4 個數字卡片
- 缺乏整體損益顯示
- 無持股部位彙總
- 缺乏視覺化趨勢

**建議新增儀表板元件**：

#### A. 總覽卡片組
```
┌────────────────┬────────────────┬────────────────┬────────────────┐
│   💰 總投入    │   📈 總市值    │   💵 總損益    │   📊 報酬率    │
│   $45,000      │   $52,300      │   +$7,300      │    +16.2%      │
│                │                │   ▲ +8.5%      │                │
└────────────────┴────────────────┴────────────────┴────────────────┘
```

#### B. 持股部位表格
| 股票 | 股數 | 成本 | 現價 | 漲跌 | 損益 | 報酬率 |
|------|------|------|------|------|------|--------|
| TSLA | 100 | $200 | $245 | +22.5% | +$4,500 | +22.5% |
| NVDA | 50 | $450 | $520 | +15.6% | +$3,500 | +15.6% |

#### C. 歷史交易圓餅圖
- 買入 vs 賣出次數占比
- 各股票交易次數分布

#### D. 月度損益趨勢
- 每月賺 / 虧的視覺化條形圖

### 3️⃣ K 線圖表建議

**建議方案**：

#### 方案 A：Plotly.js（推薦）
```html
<script src="https://cdn.plot.ly/plotly-2.27.0.min.js"></script>
```

**優點**：
- 專業金融圖表支援（K 線、成交量）
- 互動性強（縮放、懸停提示）
- CDN 引入，無需建置
- 免費開源

**建議呈現**：
```
┌─────────────────────────────────────────────┐
│  TSLA                                        │
│  ┌─────────────────────────────────────┐    │
│  │     📈 K線圖 (可拖曳/縮放)           │    │
│  │     ~~~  MA5/MA20  ~~~              │    │
│  ├─────────────────────────────────────┤    │
│  │     📊 成交量柱狀圖                  │    │
│  └─────────────────────────────────────┘    │
│                                             │
│  [1D] [1W] [1M] [3M] [6M] [1Y] [ALL]        │
└─────────────────────────────────────────────┘
```

#### 方案 B：TradingView Lightweight Charts
```html
<script src="https://unpkg.com/lightweight-charts/dist/lightweight-charts.standalone.production.js"></script>
```

**優點**：
- 更輕量 (~40KB vs Plotly ~3MB)
- 效能更好
- 專業級 K 線渲染

#### 方案 C：簡易 SVG（若只需簡單趨勢線）
- 自己用 SVG 畫簡單的折線圖
- 適合行動裝置極簡需求
